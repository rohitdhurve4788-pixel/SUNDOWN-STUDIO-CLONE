## 🛠️ Tech Used
- HTML5
- CSS3

- Swiper.js — client carousel

---

## 📱 Mobile Responsive
Made fully mobile responsive from scratch — no tutorials, no AI help.
Faced and solved multiple layout challenges manually including:
- Navigation restructure for small screens
- Hero section reflow
- Project cards switching from hover-list to card layout
- Footer layout adjustment
- Blob/animation repositioning

---

## ✨ Features
- Loader animation on page load
- Smooth scroll with Locomotive Scroll
- Infinite moving text marquee (pure CSS)
- Hover image preview on project list
- Footer reveal effect (fixed + z-index trick)
- Animated orange blob backgrounds (pure CSS)
- Swiper carousel for client logos

- Making layouts responsive without any framework
